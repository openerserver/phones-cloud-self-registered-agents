# 安全政策

感谢你帮助保护 Phones Cloud 用户和自注册被控端的安全。

## 当前支持范围

本仓库目前公开的 Windows Agent v0.1.0、Mac Agent v0.2.7 和 Android Tunnel Client v1.1.1 均处于测试阶段。安全修复会优先提供给当时仍受支持的最新测试版本。

## 私下报告安全问题

请不要为安全漏洞创建公开 Issue，也不要在公开讨论中粘贴利用代码、访问凭据、设备标识或未经处理的完整日志。

请发送邮件至 [a@phones-cloud.cn](mailto:a@phones-cloud.cn)，邮件主题建议使用：

`[Security] 平台 / 客户端版本 / 问题摘要`

报告中请尽量包含：

- 受影响的客户端、版本和操作系统版本；
- 问题描述、可能影响和最小复现步骤；
- 已进行脱敏的日志、截图或网络请求信息；
- 你已经采取的临时缓解措施；
- 便于后续沟通的联系方式。

不要直接发送密码、访问令牌、私钥或用户隐私数据。如果确有必要，请先联系我们确认安全的传输方式。

## 建议报告的问题

- 未经授权的设备注册或远程控制；
- 身份验证、会话或权限绕过；
- 本地提权、任意代码执行或敏感信息泄露；
- 更新、安装包签名或下载链路中的完整性问题；
- 可能影响其他用户、设备或服务端数据的漏洞。

## 不适合公开安全渠道的问题

普通安装失败、兼容性问题和功能建议可使用 [GitHub Issues](https://github.com/openerserver/phones-cloud-self-registered-agents/issues)。账号、订单、付款或需要提交隐私数据的问题，请使用[官网联系页面](https://www.phones-cloud.cn/contact)。

## 协调披露

收到报告后，我们会先确认影响范围并与你保持沟通。请在双方完成修复和披露计划前避免公开漏洞细节，以减少对测试用户的风险。
